# Secure Password Generator

A modern, responsive React app for generating secure and unique passwords. Customize password length, character types, and get real-time strength feedback. Features one-click copy, regenerate, and smooth UI animations.

## Features
- Cryptographically secure password generation
- Customizable options (length, uppercase, lowercase, numbers, symbols)
- Real-time password strength indicator
- One-click copy to clipboard
- Regenerate password button
- Responsive, accessible, and modern UI (Tailwind CSS)
- Visual feedback and smooth transitions

## Getting Started

1. **Install dependencies:**
   ```sh
   npm install
   ```
2. **Start the development server:**
   ```sh
   npm run dev
   ```

## Folder Structure
- `src/components` – Reusable UI components
- `src/hooks` – Custom React hooks
- `src/utils` – Utility functions
- `src/styles` – Tailwind CSS and custom styles

## Accessibility & Security
- Accessible color contrast and keyboard navigation
- Passwords generated using `window.crypto.getRandomValues`
